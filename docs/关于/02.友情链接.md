### 友链格式

互相添加友链信息，格式：(点击代码块右上角一键复制)

```yaml
- name: 俊劫的学习基地 # 昵称
  desc: 一个前端学习资源很多的小混子 # 介绍
  avatar: http://alexwjj.github.io/img/logo.png # 头像
  link: http://alexwjj.github.io/ # 链接
  bgColor: "#CBEAFA" # 可选，背景颜色
  textColor: "#6854A1" # 可选，文本颜色
```

申请前记得先添加本站，`(*╹▽╹*)`。
